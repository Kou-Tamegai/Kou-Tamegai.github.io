言語 - Language -

HTML
ホームページなどを見て、HTMLのレイアウトを大まかに再現することが出来ます。
CSS
HTML同様、ホームページのレイアウトを大まかに再現することが出来ます。またYouTubeでサムネイルの作成を何度も手がけており、視聴者を惹きつける見た目のデザインに仕上げることが出来ます。
Javascript
このポートフォリオサイトのように、HTMLとCSSでレイアウトしたサイトに動きをつけることが出来ます。
React
ReactとAPIを連携してポケモン図鑑を作ったり、firebaseを利用したGoogle認証機能、firestoreを使い簡易的な掲示板サイトなどを作成できます。
